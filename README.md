# 国内VPSリファレンス

本リポジトリは、国内事業者の VPS サービスを対象としたリファレンスです。

## 初期比較対象（5社）

- さくらのVPS
- ConoHa VPS
- XServer VPS
- WebARENA Indigo
- KAGOYA CLOUD VPS

## 初期版の対象外

- AWS
- Azure
- GCP
- Vultr
- Akamai Cloud（Linode）
- DigitalOcean
- OCI
- Hetzner

## 方針

- 国内事業者を優先
- 公式情報を第一ソースとする
- 推測は禁止（不明な項目は `unknown` を維持）
- VPS と Cloud を混同しない
